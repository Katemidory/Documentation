    Нумерованные списки

    Неважно, какие именно числа вы напишете: Markdown пронумерует список автоматически: 1. Первый пункт 2. Второй пункт 3. Третий пункт

    2. Второй пункт

    3. Третий пункт

    5. Второй пункт

    6. Третий пункт

    7. Первый пункт

    Вложенные

    Чтобы создать вложенный список, нужно поставить перед его пунктами табуляцию или несколько пробелов. В Markdown одна табуляция соответствует четырём пробелам. Список одного вида можно вкладывать в любой другой.

    1. Пункт

      1. Подпункт
        1. Подподпункт

    2. Пункт

      • Подпункт
        • Подподпункт

    3. Пункт

      • Подпункт
        • Подподпункт

    4. Пункт

      1. Подпункт

    5. Пункт

    6. [x] Отмеченный подпункт
    7. [ ] Неотмеченный подпункт
